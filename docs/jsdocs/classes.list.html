Breakpoints"after.ft.breakpoints"

The after.ft.breakpoints event is raised after the breakpoint has changed and the UI is redrawn and is supplied both the "new" current breakpoint and the previous one that was replaced.
Parameters:
Name Type Description
e jQuery.Event The jQuery.Event object for the event.
ft FooTable.Table The instance of the plugin raising the event.
current FooTable.Breakpoint The current breakpoint.
previous FooTable.Breakpoint The breakpoint that was just replaced.

Breakpoints"before.ft.breakpoints"

The before.ft.breakpoints event is raised if the breakpoint has changed but before the UI is redrawn and is supplied both the current breakpoint and the next "new" one that is about to be applied. Calling preventDefault on this event will prevent the next breakpoint from being applied.
Parameters:
Name Type Description
e jQuery.Event The jQuery.Event object for the event.
ft FooTable.Table The instance of the plugin raising the event.
current FooTable.Breakpoint The current breakpoint.
next FooTable.Breakpoint The breakpoint that is about to be applied.
